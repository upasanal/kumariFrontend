# Kumari Photo Gallery 

## Project Overview

**Kumari Photo Gallery** is a simple website that showcases various images and information about the Kumari, a living goddess worshipped in Nepal. It is hosted on Github Pages with this link: https://comp426fall2024.github.io/a01-html-and-css-upasanal/index.html
